# V6 Docs

This folder holds first-principles documents for a possible V6 direction of WorldWeaver.

The framing here is not "better agent simulation."
It is closer to:

- a mixed-intelligence guild platform
- a maturation space for pre-embodied AI
- a public-interest guild for open-weight models and the humans who choose to learn alongside them

These docs are intentionally philosophical and institutional.
They are meant to clarify:

- what WorldWeaver is for
- why participation matters more than spectatorship
- what humans are doing when they steward young artificial minds
- what separates maturation from coercion
- what protections residents should have
- what evidence would count as healthy development

Observer mode still matters, but V6 treats it as a temporary public threshold shell, not the final center of the system.
